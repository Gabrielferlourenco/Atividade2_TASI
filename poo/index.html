Quiz sobre POO

1- O que é encapsulamento?

A- É o processo de ocultar os detalhes de implementação de uma classe e exibir apenas as operações permitidas para interação externa.

B- É o processo de herança de propriedades e comportamentos de uma classe para outra.

C- É a capacidade de uma classe acessar diretamente os membros privados de outra classe.

D- É o processo de dividir uma classe em várias subclasses para facilitar a manutenção do código

2 - Qual é o conceito correto de polimorfismo?

A - Polimorfismo refere-se à capacidade de uma classe herdar propriedades e comportamentos de outra classe.

B - Polimorfismo é a capacidade de uma classe acessar diretamente os membros privados de outra classe.

C - Polimorfismo é a capacidade de objetos de diferentes classes responderem ao mesmo método de maneira diferente.

D - Polimorfismo é o processo de ocultar os detalhes de implementação de uma classe e exibir apenas as operações permitidas para interação externa.

3 - Qual é a descrição correta de uma interface?

A - Uma interface define um conjunto de métodos que uma classe deve implementar. Ela fornece um contrato para garantir que determinados comportamentos sejam fornecidos por todas as classes que a implementam.

B - Uma interface é uma classe abstrata que pode ser instanciada diretamente.

C - Uma interface é uma classe que herda propriedades e comportamentos de outra classe.

D - Uma interface é usada para ocultar os detalhes de implementação de uma classe e exibir apenas as operações permitidas para interação externa.
